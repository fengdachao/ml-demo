# 智能新闻搜索智能体 🤖

一个基于AI技术的智能新闻搜索平台，能够根据用户输入智能搜索相关新闻，并提供深度的相关新闻推荐功能。

## ✨ 主要功能

### 🔍 智能新闻搜索
- **关键词搜索**: 输入任意关键词，智能匹配相关新闻
- **多源整合**: 整合多个新闻源，提供全面的新闻覆盖
- **实时更新**: 获取最新的新闻资讯
- **智能去重**: 自动过滤重复和相似新闻

### 📊 新闻分析
- **分类统计**: 按新闻分类进行统计分析
- **来源分析**: 展示新闻来源分布
- **时间分析**: 显示最新新闻发布时间
- **数据可视化**: 直观的统计图表展示

### 🔗 相关新闻推荐
- **智能关联**: 基于选中新闻智能推荐相关内容
- **关键词提取**: 自动提取新闻关键词进行深度搜索
- **内容扩展**: 提供更多相关背景和延伸阅读

### 🎨 现代化界面
- **响应式设计**: 适配各种设备屏幕
- **美观的UI**: 现代化的界面设计
- **流畅交互**: 优化的用户体验
- **实时反馈**: 搜索过程中的实时状态提示

## 🚀 快速开始

### 环境要求
- Python 3.7+
- 现代浏览器 (Chrome, Firefox, Safari, Edge)

### 安装运行

1. **克隆项目**
   ```bash
   git clone <项目地址>
   cd news-search-agent
   ```

2. **一键启动**
   ```bash
   ./start.sh
   ```

3. **手动启动**
   ```bash
   # 安装依赖
   pip3 install -r requirements.txt
   
   # 启动应用
   python3 app.py
   ```

4. **访问应用**
   打开浏览器访问: `http://localhost:8000`

## 📖 使用指南

### 基础搜索
1. 在搜索框中输入关键词（如：人工智能、新能源、区块链等）
2. 选择希望显示的结果数量
3. 点击"搜索新闻"按钮
4. 查看搜索结果和统计信息

### 深度探索
1. 在搜索结果中点击任意新闻卡片查看详情
2. 点击"相关"按钮搜索相关新闻
3. 在新闻详情模态框中点击"查找相关新闻"
4. 浏览相关新闻获取更全面的信息

### 高级功能
- **搜索历史**: 自动保存搜索历史，快速重复搜索
- **结果筛选**: 按分类、来源、时间等维度筛选
- **数据导出**: 支持搜索结果的数据导出
